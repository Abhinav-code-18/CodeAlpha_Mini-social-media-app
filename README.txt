
# Mini Social Media App (MongoDB Version)

## Features
- Express.js + MongoDB backend (Mongoose)
- Frontend with improved styling (cards, padding, colors)
- Demo users and posts auto-seeded
- "Act as user" dropdown instead of login

## Setup Instructions
1. Install Node.js and MongoDB.
2. Start MongoDB (default port 27017).
3. Extract this ZIP, open terminal in folder.
4. Run:
   ```bash
   npm install
   npm run seed
   npm start
   ```
5. Open `http://localhost:3000` in your browser.
